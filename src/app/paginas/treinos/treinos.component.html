Exercícios de Barra e Flexão

Exercício de Barra

A barra fixa é um exercício muito eficaz para fortalecer a parte superior do corpo, especialmente as costas e os braços. Ao realizar a barra, é importante manter a postura correta e os ombros relaxados, evitando usar a força das pernas. Para iniciantes, pode-se começar com a barra assistida ou com elásticos para suporte adicional.

Exercício de Flexão

As flexões são um dos exercícios mais comuns para trabalhar o peitoral, tríceps e ombros. Para realizar uma flexão corretamente, é importante manter o corpo alinhado da cabeça aos pés e não deixar os quadris caírem. Variantes de flexão, como as flexões inclinadas ou com os pés elevados, podem ser feitas para intensificar o treino.
